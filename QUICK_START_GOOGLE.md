# Quick Start: Get Your Site in Google Search

## üéØ 3-Step Quick Start (Do This First!)

### Step 1: Add Google Search Console Verification (5 minutes)

1. Go to: https://search.google.com/search-console
2. Click "Add Property"
3. Enter: `https://auditra-web.web.app`
4. Choose "HTML tag" verification
5. Copy the meta tag (looks like: `<meta name="google-site-verification" content="...">`)
6. Send me the content value and I'll add it to your site

OR

7. Choose "HTML file upload"
8. Download the file (e.g., `google1234567890abcdef.html`)
9. Save it in `frontend/public/` folder
10. I'll help you deploy it

### Step 2: Submit Your Sitemap (2 minutes)

After verification:
1. In Search Console, click "Sitemaps" (left menu)
2. Enter: `https://lioncarsa.vercel.app/sitemap.xml`
3. Click "Submit"

### Step 3: Request Indexing (1 minute)

1. Use the URL Inspection tool (top search bar)
2. Enter: `https://auditra-web.web.app`
3. Click "Test Live URL"
4. Click "Request Indexing"
